== KPCC Bulletin Board

This is a bulletin board that makes use of KPCC's AudioVision API
https://github.com/SCPR/api-docs/tree/master/AudioVision/v1


== Getting Started

1. Change directory to <tt>kpcc-bulletin</tt> and start the web server:
       cd kpcc-bulletin; rails s

2. Go to http://localhost:3000/

3. Enter in keywords into the form inputs to either add articles or remove articles from the bulletin.
